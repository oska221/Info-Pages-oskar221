ZIP ↔ PNG Converter with Color Mapping

Turn your ZIP files into PNG images and back again.
This tool uses a special color mapping system to safely store ZIP file data inside PNG pictures. It comes with a simple GUI (Tkinter), so anyone can use it easily.

Overview

With this program you can:

  • Encode .zip files into .png images.
  • Decode .png images back into .zip files.
  • Use a random color map for hexadecimal digits (0–f).
  • Add black separator lines every ~2MB for safety.
  • Run it either directly from Python files or as a standalone .exe.

How It Works

!! WARNING !!

The generated PNG file can be much larger than the original ZIP — in some cases up to 15× the original size — due to the encoding method and black separator lines. Do NOT modify the PNG pixels, otherwise the ZIP file may become unrecoverable!

Encoding ZIP → PNG

  1. Read your ZIP file as bytes.
  2. Add metadata:
    • Length of file extension (like .zip).
    • File extension itself.
  3. Convert all bytes into a hexadecimal string.
  4. Make a random color map for hex digits (0–f).
  5. Create the PNG picture:
    • First 16 pixels store the color map.
    • Each hex character becomes a colored pixel.
    • Black separator lines are added every ~2MB to prevent problems.
    • Pad remaining pixels to make a rectangle.
  6. Save the PNG file.

This way, the PNG stores everything needed to get back your original ZIP.

Decoding PNG → ZIP

  1. Open the PNG and read all pixels.
  2. Take the first 16 pixels as the color map.
  3. Skip black separator lines.
  4. Convert pixels back to hex characters.
  5. Turn hex string back into bytes.
  6. Read metadata to get the original file extension.
  7. Save the decoded ZIP file in the folder you choose.

Requirements

  • Python 3.10 or newer
  • Pillow library (for working with images)

pip install Pillow
